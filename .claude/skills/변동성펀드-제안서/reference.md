# reference.md — 수식 규약 & 제안서 슬라이드 구조

## 1. 옵션 수학 규약 (시뮬레이터 내부)

### 1.1 Black-Scholes (유러피언, 배당 q)
```
d1 = [ln(S/K) + (r − q + σ²/2)·T] / (σ√T),   d2 = d1 − σ√T
Call = S·e^{−qT}·N(d1) − K·e^{−rT}·N(d2)
Put  = K·e^{−rT}·N(−d2) − S·e^{−qT}·N(−d1)
Δcall = e^{−qT}·N(d1),   Δput = e^{−qT}·(N(d1)−1)
Γ = e^{−qT}·n(d1)/(S·σ√T)
Vega = S·e^{−qT}·n(d1)·√T        (/100 → 1%p 단위)
Θ, ρ 는 표준식 (Θ는 /365 일단위, ρ는 /100 1%p)
```
`N`=표준정규 CDF(Abramowitz-Stegun erf 근사), `n`=pdf. 타입 코드 **`'c'` / `'p'`**.

### 1.2 배리어 (다운아웃 풋 + 리베이트) — Reiner-Rubinstein
`down-out put = vanilla put − down-in put`. down-in put(φ=−1, η=+1):
- K>H: `B − C + D`,  K<H: `A`  (A,B,C,D는 표준 배리어 항, b=r−q).
- 이미 S≤H면 0. 그릭스는 중심차분(FD); vega/rho는 대시(—) 표기.
- **리베이트(낙아웃 시 지급, 만기지급)**: 이론가 = DownOutPut + `R·e^{−rτ}·P_hit`. 하향 first-passage 확률
  `P_hit = N((ln(H/S)−ν·τ)/(σ√τ)) + (H/S)^{2ν/σ²}·N((ln(H/S)+ν·τ)/(σ√τ))`, ν=r−q−σ²/2 (S≤H면 1).
- **표시**: 애널리틱 차트는 **만기 손익 P&L**(=페이오프−프리미엄×e^{rT}=pnlUnit, 프리미엄 포함, 원금대비%)로 그림. KO 단일 카드=단일 곡선(만기 S 기준 S≤H→리베이트, S>H→max(K−S,0)). 합성 카드만 도달/미도달 2곡선. 동일행사가 풋매도+KO풋매수는 미도달 시 (풋프리미엄−KO프리미엄) 상수. MC=낙아웃 비율.

### 1.3 풋 매도 행사가
행사가 K는 **S₀ 대비 %로 직접 지정**(OTM은 K<100%). 목표 프리미엄 역산 개념은 사용하지 않는다. S₀ 기본값=100.

### 1.4 몬테카를로 Dynamic Hedging
```
경로: S_{t+dt} = S_t·exp[(μ − σ²/2)dt + σ√dt·Z],  μ = (r−q) 위험중립 기본
복제포트 초기: 가치=이론가 V₀, 현물 Δ₀단위 + 현금
매 스텝: cash *= e^{r·dt};  cash += h·S·q·dt (배당);  리밸런싱 시 Δh 거래 + 비용(bps)
만기: port = h·S_T + cash;  실제페이오프 actual = payoff(S_T, alive)
트래킹오차 = (port + 누적거래비용) − actual   → 순수 트래킹오차(평균≈0)
MC 복제가치 = mean(e^{−rT}·actual)  ≈ 이론가 V₀ (정합 검증)
배리어: 일별 모니터링 → 다운아웃 값 MC ≥ 연속식(정상)
```
부호: 숏풋 V₀·MC복제가치 음수. UI "수취 프리미엄" +표기. 프리미엄 확보확률=만기 세전 페이오프수익률≥수취 프리미엄.
**동적헤지 수익률(제안서 산점도용)**: `dhRet = pnlUnit − 거래비용 + 복제오차(hedgePnl)` (경로의존→클라우드). 산점도 X=만기 기초자산(S₀=100), Y=dhRet(합성=Σ전략), 점=경로. 배리어 있으면 **①미터치·②터치 2개로 분리**(경로별 touched). **터치 즉시 KO 델타=0 청산**(freq 무관) → 터치 경로는 숏풋 하방 노출.

## 3. 제안서 슬라이드 구조 (`변동성매매자문_20260513.pptx` 기준, 14슬라이드)
1. 표지 — 상품명·운용부문·연월·컴플라이언스 확인필 문구
2. 상품 개요 — 명칭/개요/타겟수익률/관찰주기/운용전략/유형/투자등급/최저금액/운용대상/수수료/환매 + 랩 disclaimer 5줄
3. 투자 포인트(목차) — 전략소개 / 시뮬레이션·성과 / 운용부서
4. 투자포인트 요약 — 현물 델타조절로 복제 / 모델기반 / 타겟프리미엄 / 정량헤지
5. 전략소개① 페이오프 — 회차별 목표(5%→10%) 흐름, 상방 cap·하방 손실
6. 전략소개② 편입비 로직 — 주가·변동성별 편입비 조정(Buy Low/Sell High)
7. 전략소개③ 일별 프로세스 — 변동성예측→행사가 역산→델타→리밸런싱
8. 전략소개④ 장단점·이론 — 장점/리스크 + BS 동적복제 이론 + 직접매도 vs 복제 비교표
9. 시뮬레이션 트랙레코드 — 기간·시뮬수·목표달성률(전체/6M이내)·산점도
10. 단일회차 사례(달성) — 설정/달성일·소요일·매매횟수·운용성과 vs 주식수익률
11. 단일회차 사례(미달) — 동일 포맷
12. 운용 시스템화 — DB-Backend-Frontend, 대시보드
13. 부서 Mission·운용현황 — AUM 표
14. 대표펀드 성과표

**규칙**: 9~11의 수치는 백테스트(작업 C)에서, 마케팅 포인트는 사용자 제공. "상기 자료는 이해를 돕기 위한 예시" 캡션과 컴플라이언스 문구 유지.
