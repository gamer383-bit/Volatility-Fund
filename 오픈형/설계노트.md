# 오픈형 변동성 하베스트 레버리지 — 설계 노트 (v1, 2026-07-16)

단위형(변동성 하베스트 펀드)과 같은 취지의 **오픈형 레버리지 2종**. BM=기초자산 150%(KOSPI200 / 삼성전자 / SK하이닉스 각각), 편입비 100~200%, **상승 시 편입비↓·하락 시 편입비↑**(레버리지 ETF 순응매매의 정반대), 주기적으로 150% 복귀.

> **설계 가정 (v1): 현 고변동 국면을 반영해 지수 변동 범위를 ±40%까지 가정** (±20%가 아님). 편입비 캡(100/200%)이 ±40% 부근에서 도달하도록 설계.

## 상품 정의

### 상품1 — 룰베이스 레버리지
- 편입비 `w = 150% − k × (지수/기준지수 − 1)`, 캡 [100%, 200%]
- **k=1.25 (표준)**: 지수 ±10% → 편입비 ∓12.5%p, **±40%에서 캡 도달** ← v1 설계 가정
- k=2.00 (공격, 비교용): ±10% → ∓20%p (레버리지 ETF 미러), ±25%에서 캡 도달
- 장점: 단순·투명·설명 용이. 단점: 기울기(k)가 상수라 변동성 수준 반영 없음

### 상품2 — 옵션복제 레버리지 (고정만기)
- 편입비 `w = 100% + ATM급 풋매도 복제비중(행사가 K, 만기 T=0.25년 고정)`
- **만기 고정(constant maturity)** → 시간 경과에 따른 감마 폭주·핀 리스크 없이 편입비 곡선이 항상 동일 (단위형 시뮬레이터의 만기접근 문제 해결)
- 행사가 K는 **리밸런싱일에 편입비 150%가 되도록 역산**: σ50% → K=103.6, σ60% → K=105.1
- σ50~60%·T0.25 조합은 **±40% 구간에 걸쳐 자연스럽게 100~200%를 커버** (60에서 195~198%, 140에서 109~113%) — v1 가정과 정합. 캡은 구조적으로 자동 충족

## 실제 순매수 — 목표 편입비 vs 자체 드리프트

지수가 움직이면 보유 주식 평가액과 NAV가 함께 변해 **편입비가 스스로 움직인다(드리프트)**. 실제 매매 = 목표 − 드리프트.
- 드리프트 편입비(리셋 w=150%에서 지수 m으로 이동): `w_d = 1.5(1+R)/(1+1.5R)`, R=m/100−1
- **레버리지 영역(w>1)에서는 상승 시 NAV가 노출보다 빨리 늘어 편입비가 자연 하락** → 목표 방향과 같음 → **실제 매매는 목표 변화의 약 절반**

| 지수 | 드리프트 편입비 | 상품1 목표→**실제 순매수** | 상품2 σ50 목표→**실제 순매수** | 레버리지ETF 순매수 |
|---:|---:|---:|---:|---:|
| 60 | 225.0 (캡 초과) | 200.0 → **−25.0** (캡 준수 매도) | 197.5 → −27.5 | −400.0 (NAV 붕괴) |
| 70 | 190.9 | 187.5 → **−3.4** | 191.9 → +1.0 | −150.0 |
| 80 | 171.4 | 175.0 → **+3.6** | 181.2 → +9.7 | −66.7 |
| 90 | 158.8 | 162.5 → **+3.7** | 166.2 → +7.4 | −25.0 |
| **100** | 150.0 | 150.0 → 0 | 150.0 → 0 | 0 |
| 110 | 143.5 | 137.5 → **−6.0** | 135.2 → −8.3 | +16.7 |
| 120 | 138.5 | 125.0 → **−13.5** | 123.4 → −15.1 | +28.6 |
| 140 | 131.2 | 100.0 → **−31.2** | 109.0 → −22.3 | +44.4 |

(변동 후 NAV 대비 %p · '지수 m까지 한 번에 이동' 기준 — 일별 리밸런싱 시 소폭 분할 집행)

**핵심 발견**
1. **+10% 상승 시 실제 순매도는 −6%p** (목표 변화 −12.5%p의 절반) — 드리프트(150→143.5)가 절반을 알아서 처리
2. **−10% 하락 시 실제 순매수는 +3.7%p뿐** — 하락 시에도 드리프트가 편입비를 올려주므로(150→158.8) 실탄 소모 작음
3. **순매수 최대 구간은 −15~−20%(+3.6~+4.2%p)**, 그 아래로는 매수량이 줄다가 **−30% 이하에선 캡(200%) 준수를 위해 오히려 매도** — 급락 시 "떨어지는 칼을 무한정 잡지 않는" 자기제어 구조
4. 레버리지 ETF와 규모 비교: ±10%에서 ETF ±17~25%p vs 본 펀드 ∓4~8%p — **시장충격이 훨씬 작음**

## 오픈형 150% 복귀(리셋) — 스무딩 방안 (1-4)

한 번에 리셋하면 편입비가 점프하므로:

1. **트랜치 롤링 (권장)** — 오버레이를 N개 트랜치로 분할, 매주 1/N씩 기준 리셋.
   - 상품2: T=0.25(13주) → **13개 주간 트랜치**, 매주 1/13씩 행사가를 현재 지수 기준 재설정. 편입비 = 13개 행사가의 평균 델타 → 연속적, 일시 매매 ≤ 리셋폭/13
   - 상품1: 기준지수 S₀를 13주 트랜치 평균으로 운용 (동일 원리)
2. 선형 수렴 — 리셋일에 150%와의 차이를 n영업일(예 5일)에 걸쳐 1/n씩 축소
3. 기준지수 EMA — S₀를 3개월 지수이동평균으로 상시 대체 (가장 부드러우나 리셋일=150% 미보장)

→ **권장: 주간 트랜치 롤링** (설정/해지 자금 유출입도 트랜치 단위로 흡수)

**해석 (트레이드오프 정직 고지)**
- 보합~완만한 등락에서 두 상품 모두 **BM150 대비 우위** (저가매수·고가매도 수확 → 승률 43%→47~48%)
- **급락 꼬리에서는 BM보다 손실 확대** (하락 시 편입 증가의 본질적 비용), 급등 꼬리에선 덜 참여
- "꼬리를 팔아 몸통을 사는" 구조 — 단위형 변동성 하베스트와 동일 철학의 오픈형 구현
- k=1.25(±40% 설계)는 k=2 대비 꼬리 손실 소폭 완화 (VaR −55.5%→−53.2%)

## 시장 안정화 스토리 (마케팅 연결)
레버리지 ETF(국내외 약 40조)가 ±10%일에 종가에서 약 8조를 **순응매매**하는 시장에서, 본 오픈형은 같은 날 **역방향으로 매매**한다. AUM이 커질수록 레버리지발(發) 되먹임 변동성을 흡수하는 **시장 안정화 기여** 상품.
